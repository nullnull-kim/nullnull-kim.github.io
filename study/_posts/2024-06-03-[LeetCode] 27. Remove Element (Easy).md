# [LeetCode] 27. Remove Element (Easy)

[문제로](https://leetcode.com/problems/remove-element/){: target="_blank"}

## 내가 푼 것 
```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int count = 0;
        for(int i = 0; i < nums.length-1; i ++) {
            if(nums[i] != val) {
                nums[count++] = nums[i];
            }
        }

        return count;
    }
}
 
## 답안1
```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int index = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != val) {
                nums[index] = nums[i];
                index++;
            }
        }
        return index;
    }
}
